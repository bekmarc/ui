## Usage

<Snippet>```import {CodeBlock} from "@medusajs/ui"```</Snippet>

<Snippet>
  ```
  <CodeBlock 
    snippets={[
      { 
        language: "tsx", 
        label: "Label", 
        code: 'import { useProduct } from "medusa-react"'
      },
    ]}
  >
    <Codeblock.Header />
    <CodeBlock.Body />
  </CodeBlock.Body>
  
  ```
</Snippet>
