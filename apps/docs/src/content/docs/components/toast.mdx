## Usage

Add the Toaster component somewhere in your tree

<Snippet>```import {Toaster} from "@medusajs/ui"```</Snippet>

<Snippet>
  ```
  <Toaster />
  ```
</Snippet>

The <Code>useToast</Code> hook returns a <Code>toast</Code> function that you can use to display a toast.

<Snippet>```import {useToast} from "@medusajs/ui"```</Snippet>

<Snippet>
```
const { toast } = useToast()

return (

  <Button 
    onClick={() => 
      toast({ 
        title: "Toast title",
        description: "Toast body",
      })
    }
  >
    Trigger
  </Button>
)
```
</Snippet>
