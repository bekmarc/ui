## Usage

A Wizard is a component that guides a user through a set of steps to complete a larger task. An example of a use case for a Wizard is a multi-step form. The Wizard component is built using the Focus Modal and Progress Tabs components.

## Implementation

The Wizard component can be built using the Focus Modal and Progress Tabs components. The Modal component is used to display the Wizard and the Progress Tabs component is used to manage the steps of the Wizard.

<Snippet>
```import {FocusModal, ProgressTabs} from "@medusajs/ui"```
</Snippet>
